# RWET
Program error reporting is always crushing, what if you make it cute? This week I tried to use pyhton's error reporting as a format and combine it with the content of children's picture books to generate cute error message.

The format of error reporting I used is:

> []Error: [Error Message]

> [Description of the error, including any relevant information such as the invalid input value]

> [Suggestions for resolving the error or troubleshooting steps to take]
